# Kreuzer Trockenbau & Malerbetrieb – Website

Moderne, dunkle Website für Kreuzer Trockenbau & Malerbetrieb, Haibühl im Bayerischen Wald.

## Tech-Stack

- [Astro](https://astro.build/) v5 – Static Site Generator
- [Tailwind CSS](https://tailwindcss.com/) v4 – Styling
- TypeScript

## Entwicklung

```bash
# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev

# Für Produktion bauen
npm run build

# Produktions-Build testen
npm run preview
```
